## What's Included

- Pre-configured folder structure for ROMs and system files
- Obtainium APK for app management (automatically updated via workflows)
- Turnip GPU drivers 
- ES-DE custom system configurations (automatically updated via workflows)

## Automated Workflows

This repository uses GitHub Actions to automatically update:
- ES-DE system configurations
- Obtainium APK releases

The intention is that this Quick Start Pack will be integrated with an automated setup tool that will:
- Download and extract this folder structure
- Install and configure emulators automatically (as far as possible)
- Guide users through the remaining manual steps
